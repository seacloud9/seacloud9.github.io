Why webGL?

  • Ubiquitous
  • GL (graphics language) is powerful
  • You only need a browser and a notepad, no need for expensive software

BabylonJS What is it?

  • BabylonJS was created by David Cathue / DeltaKosh at Microsoft
  • It is a performant webGL engine
  • Enables developers to rapidly create interactive 3D content
  • What was babylonJS written in? TypeScript
  • Fast and it is easy to create a project with it
  • Has excellent documentation
  • How is it different from threeJS?
    • No Canvas Fallback
    • No CSS renderer
    • Basic Collisions baked in
    • Audio Analyzer Class included
    • Physics made easy
